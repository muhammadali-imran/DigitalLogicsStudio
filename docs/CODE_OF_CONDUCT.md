# Code of Conduct

## Our Pledge

We want Boolforge to be a respectful, learning-friendly project for contributors, students, maintainers, and reviewers.

## Expected Behavior

- Be kind, direct, and constructive.
- Explain technical feedback with context.
- Respect different experience levels.
- Keep discussions focused on project quality and user impact.
- Credit contributors for their work.
- Ask questions when requirements are unclear.

## Unacceptable Behavior

- Harassment or personal attacks.
- Discriminatory language or behavior.
- Publicly sharing private information.
- Repeatedly derailing issues or pull requests.
- Threatening, abusive, or insulting comments.

## Enforcement

Maintainers may edit or remove comments, close discussions, request behavior changes, temporarily restrict participation, or permanently ban contributors in severe or repeated cases.

## Reporting

Report conduct concerns privately to project maintainers through the repository owner contact channels. Include relevant links and context.

## Scope

This code applies to repository activity, issues, pull requests, discussions, community channels, and project-related events.
