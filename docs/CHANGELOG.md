### Added

- React 18 application built with Create React App.
- Digital-logic topic pages and interactive tools.
- React Router route map with lazy-loaded pages.
- Login, signup, logout, session restore, and protected profile page.
- Backend API integration with Axios.
- Progress service for problems, topics, activity, and profile snapshots.
- SEO metadata catalog, sitemap/robots generation, and prerender support.
- Vercel analytics integration.
